# 🎞️ GIF Creator using Python

A simple Python project to generate animated GIFs from images using **imageio**.

---

## 📂 Project Structure

```bash
Creating-gifs/
│
├── Code/
│   ├── create_gif.py
│   ├── create-gif-2.py
│
├── pics/
│   ├── camera-1.png
│   ├── camera-2.png
│   ├── team-pic1.png
│   ├── team-pic2.png
│
├── Result/
│   ├── camer-click.gif
│   ├── team.gif
│
└── README.md
```

---

## 🚀 Features

* Convert multiple images into GIFs
* Supports multiple image sets (camera, team, etc.)
* Customizable speed and looping
* Organized folder structure

---

## 🛠️ Requirements

Install dependencies:

```bash
pip install imageio pillow numpy
```

---

## ▶️ How to Run

1. Navigate to the project folder:

```bash
cd Creating-gifs/Code
```

2. Run the script:

```bash
python create_gif.py
```

3. Output GIF will be saved in:

```bash
Result/
```
